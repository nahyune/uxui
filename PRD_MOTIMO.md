## 1. 제품 개요

**제품명:** MOTIMO
**슬로건:** 아끼고, 기록하고, 함께 바꾸는 소비 습관 챌린지
**플랫폼:** 모바일 앱 (iOS / Android)
**대상 사용자:** 소비 습관을 개선하고 싶은 20–30대, 함께 절약 챌린지에 참여하고 싶은 사용자

### 핵심 가치 제안

| 가치 | 설명 |
|------|------|
| 함께 | 혼자 하기 어려운 절약을 챌린지로 달성. 참여자 수와 채팅으로 사회적 동기부여 |
| 기록 | 소비를 기록하면 패턴이 보이고, 카테고리별 통계로 지출 습관 인식 |
| 변화 | 주간 리포트와 랭킹으로 절약 행동의 성취감을 수치화 |

### 문제 정의

- **인식 부재:** 무의식적 소비가 누적되어도 전체 규모를 모름
- **지속 실패:** 혼자 절약을 다짐해도 동기 유지가 어려움
- **피드백 부재:** 절약의 효과를 즉시 확인할 수단이 없음

### 솔루션

- 카테고리별 지출 기록 → 소비 패턴 시각화
- 공개 챌린지 참여 → 소셜 압박과 격려로 지속성 확보
- 실시간 채팅 + 랭킹 → 즉각적인 피드백과 성취감 제공

## 2. 사용자 페르소나

### 페르소나 A — 절약 입문자 "지출 많은 직장인"

- **나이/직업:** 25세, 사회초년생
- **상황:** 월급이 들어오면 어디 썼는지 모르게 사라짐
- **불편:** 가계부 앱은 혼자 쓰다 3일 만에 포기
- **기대:** 친구들이랑 같이 하면 끝까지 할 수 있을 것 같음
- **핵심 니즈:** 간편한 기록, 사회적 동기부여

### 페르소나 B — 목표 중심 절약가 "재테크 관심 대학원생"

- **나이/직업:** 28세, 대학원생
- **상황:** 매달 저축 목표가 있지만 카페·쇼핑 지출이 넘침
- **불편:** 지출 카테고리가 어디서 새는지 파악하기 어려움
- **기대:** 카테고리별 통계와 주간 리포트로 구체적인 개선 포인트 파악
- **핵심 니즈:** 통계·분석, 챌린지 달성률 추적

## 3. 사용자 흐름

```
[최초 진입]
온보딩 슬라이드 1 → 온보딩 슬라이드 2 → 로그인

[메인 루프]
홈 (대시보드)
 ├── 소비체크 탭
 │    ├── 오늘 지출 현황 조회
 │    ├── 소비 올리기 → 항목 추가 → 채팅방 공유(선택)
 │    └── 소비 통계 / 주간 리포트 조회
 ├── 챌린지 탭
 │    ├── 나의 챌린지 목록 → 챌린지 상세(참여 중)
 │    ├── 인기 챌린지 탐색 → 챌린지 상세(참여 전) → 참여하기
 │    └── 새 챌린지 만들기
 ├── 채팅 탭
 │    ├── 채팅 목록
 │    ├── 일반 채팅방
 │    └── 챌린지 채팅방 → 소비 공유 → 채팅방 선택(Checkbox)
 ├── 알림 탭
 │    └── 알림 목록 (D-day / 랭킹 / 미기록 / 리포트)
 └── 마이 탭
      ├── 프로필 / 소비 리스트 / 친구 관리
      ├── 설정 → 소비 기록 목록
      └── 챌린지 설정 → 랭킹
```

### 5.4 소비 올리기 (Add Spending)

**목적:** 지출 항목을 카테고리별로 기록

**입력 필드:**

| 필드 | 타입 | 유효성 검사 |
|------|------|-------------|
| 카테고리 선택 | 수평 스크롤 탭 | `전체` `식비` `교통` `의료` `쇼핑` `카페` `여가` `생활` `기타`<br>선택: `#90D50F` 배경, 미선택: `#F6F6F6` 배경 |
| 상호명 | 텍스트 입력 | placeholder "어디서 소비했나요?", `#AFAFAF` |
| 금액 | 숫자 입력 | 숫자 키패드 자동 호출, 원화(₩) prefix |
| 날짜/시간 | 날짜 피커 | 기본값: 현재 날짜 |
| 메모 (선택) | 텍스트 입력 | placeholder "메모를 남겨보세요" |
| 저장 버튼 | Primary CTA | "저장하기", 70px, `Body1/SemiBold/24` |

**지출 항목 예시:**

| 이모지 | 상호명 | 금액 | 카테고리 |
|--------|--------|------|----------|
| ☕ | 스타벅스 | 6,100원 | 카페 |
| 🛍 | 무신사 | 62,280원 | 쇼핑 |
| 🎤 | 노래방 | 10,000원 | 여가 |
| 🍿 | CGV | 15,000원 | 여가 |
| 📦 | 쿠팡 | 9,800원 | 쇼핑 |

**저장 후 동작:** 토스트 "소비가 기록되었어요!" → 소비체크 화면 복귀
**채팅방 공유 옵션:** 저장 완료 후 "채팅방에 공유할까요?" 바텀 시트 노출

### 5.5 챌린지 (Challenge)

**목적:** 참여 중인 챌린지 관리 및 신규 챌린지 탐색

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 상단 탭 | "나의 챌린지" / "탐색" 탭 전환 |
| 나의 챌린지 카드 | 챌린지명(`Body2/Medium/22`) + 참여 인원 + D-Day + 달성률 프로그레스 바 + 기간 |
| 하단 CTA | "지금 참여하세요 🔥" → 챌린지 탐색 화면 |
| 새 챌린지 만들기 | 우상단 또는 플로팅 (+) 버튼 |

**나의 챌린지 카드 상세:**

```
┌──────────────────────────────────┐
│ 카페 금지 챌린지 ☕    [D-3]      │ ← Medium/22 / Secondary 뱃지
│ 참여자 뱃지: 83명이 함께하고 있어요!│
│ ████████░░  70%                  │ ← Primary 프로그레스 바
│ 26.04.06 ~ 04.27                 │ ← Medium/16, #434343
└──────────────────────────────────┘
```

**D-Day 뱃지 규칙:**

| 조건 | 배경 | 텍스트 |
|------|------|--------|
| D-7 이상 | `#F6F6F6` | `#686868` |
| D-3 이하 | `#FFF5E7` | `#E16500` |
| D-1 | `#FFE5E5` | `#D00000` + "내일 마감!" |

### 5.6 챌린지 상세 — 참여 전 (Challenge Detail — Join)

**목적:** 챌린지 정보 제공 및 참여 유도

| 항목 | 스펙 |
|------|------|
| 챌린지명 | `Body1/SemiBold/24`, 이모지 포함 |
| 조건 | ✓ 아이콘(Check-Circle, 15px) + `Body2/Medium/16`, `#434343` |
| 기간 | 📅 아이콘(Calendar, 15px) + `Body2/Medium/16`, `#434343` |
| 참여 인원 | 참여자 뱃지 — `Body2/Medium/16`, `#E16500`, 배경 `#FFF5E7` |
| 챌린지 소개 | 자유 텍스트, 최대 200자, `Body2/Medium/16` |
| 개설자 정보 | 프로필 이미지 + 닉네임, `Body2/Medium/16` |
| CTA | "챌린지 참여하기" — Primary 버튼, 70px, 하단 고정 |

**참여 완료 후:** 챌린지 채팅방 자동 입장 안내 바텀 시트 표시

### 5.7 챌린지 상세 — 참여 중 (Challenge Detail — Active)

**목적:** 나의 챌린지 달성 현황 확인 및 소비 체크 CTA 제공

| 항목 | 스펙 |
|------|------|
| 챌린지명 | `Body1/SemiBold/24`, 이모지 포함 |
| 나의 참여도 | 원형 프로그레스 차트 + 달성률 % (`Title/SemiBold/36`) |
| 동기부여 메시지 | "🔥 오늘 소비 체크하면 {N}% 달성!" — `Body2/Medium/16` |
| 소비 체크 현황 | "7/10 완료" — `Body1/SemiBold/20` |
| CTA (미완료) | "오늘 소비 체크하기" — Primary 버튼, 70px |
| CTA (완료) | "오늘 완료 ✓" — Disabled 버튼 (`#DDDDDD` 배경) |
| 랭킹 바로가기 | 우상단 링크 → 랭킹 화면 |

### 5.8 새 챌린지 만들기 (New Challenge)

**목적:** 사용자가 직접 챌린지를 개설

**입력 필드:**

| 필드 | 타입 | 유효성 검사 |
|------|------|-------------|
| 챌린지 제목 | 텍스트 입력 | 최소 2자, 최대 20자 |
| 목표 금액 | 숫자 입력 | 1,000원 단위, 최소 1,000원 |
| 기간 | 날짜 범위 피커 | 시작일 ≥ 오늘, 최소 3일, 최대 30일 |
| 챌린지 소개 | 텍스트에어리어 | 최소 10자, 최대 200자. 글자 수 카운터 표시 |
| 카테고리 | 단일 선택 칩 | `전체` `식비` `교통` `의료` `쇼핑` `카페` `여가` `생활` `기타` |

**저장 버튼:** 모든 필수 필드 입력 시 Primary 활성화, 미입력 시 Disabled
**개설 완료 후:** 생성된 챌린지 상세 화면으로 이동 + "챌린지가 만들어졌어요!" 토스트

### 5.9 채팅 목록 (Chat)

**목적:** 참여 중인 채팅방 목록 조회

**채팅방 카드 스펙:**

```
┌──────────────────────────────────────┐
│ [썸네일]  소비 기록 챌린지 📝  16명   │ ← SemiBold/20
│           이번 달도 다 같이 살아남아… │ ← Medium/16, Gray50, 1줄 말줄임
│                           오후 8:45  │ ← Medium/16, Gray40
│                           [미읽음 N]  │ ← Primary 원형 뱃지
└──────────────────────────────────────┘
```

**채팅방 유형:**
- 챌린지 채팅방: 챌린지 아이콘 + 이모지 썸네일
- 일반 채팅방: User-Multiple-Group 아이콘 썸네일

**화면 구성:**

| 영역 | 상세 |

| 목록 | 채팅방/친구 이름 + 인원 수 + 체크박스 |
| 하단 CTA | "보내기" — 1개 이상 선택 시 Primary 활성화 |

**다중 선택:** 최대 5개 채팅방 동시 공유 가능

| 챌린지 D-day | "소비 기록 챌린지 📝 D-1! 마지막까지 함께해요 💪" | 종료 D-3, D-1 | 챌린지 상세 |
| 랭킹 변동 | "🔥 순위가 올랐어요! 지금 5위예요" | 순위 변동 시 | 랭킹 화면 |
| 소비 미기록 | "아직 오늘 기록이 없어요 ✍️ 지금 체크하고 챌린지 이어가요!" | 당일 21시 미기록 | 소비체크 |

**알림 카드 스펙:**
- 아이콘 + 메시지 본문 (`Body2/Medium/16`) + 경과 시간 (`Body2/Medium/16`, Gray40)

- 읽음: `#FFFFFF` 배경
