## 1. 제품 개요

**제품명:** MOTIMO  
**슬로건:** 아끼고, 기록하고, 함께 바꾸는 소비 습관 챌린지  
**플랫폼:** 모바일 앱 (iOS / Android)  
**대상 사용자:** 소비 습관을 개선하고 싶은 20–30대, 함께 절약 챌린지에 참여하고 싶은 사용자

### 핵심 가치 제안

| 가치 | 설명 |
|------|------|
| 함께 | 혼자 하기 어려운 절약을 챌린지로 달성. 참여자 수와 채팅으로 사회적 동기부여 |
| 기록 | 소비를 기록하면 패턴이 보이고, 카테고리별 통계로 지출 습관 인식 |
| 변화 | 주간 리포트와 랭킹으로 절약 행동의 성취감을 수치화 |

### 문제 정의

- **인식 부재:** 무의식적 소비가 누적되어도 전체 규모를 모름
- **지속 실패:** 혼자 절약을 다짐해도 동기 유지가 어려움
- **피드백 부재:** 절약의 효과를 즉시 확인할 수단이 없음

### 솔루션

- 카테고리별 지출 기록 → 소비 패턴 시각화
- 공개 챌린지 참여 → 소셜 압박과 격려로 지속성 확보
- 실시간 채팅 + 랭킹 → 즉각적인 피드백과 성취감 제공

## 2. 사용자 페르소나

### 페르소나 A — 절약 입문자 "지출 많은 직장인"

- **나이/직업:** 25세, 사회초년생
- **상황:** 월급이 들어오면 어디 썼는지 모르게 사라짐
- **불편:** 가계부 앱은 혼자 쓰다 3일 만에 포기
- **기대:** 친구들이랑 같이 하면 끝까지 할 수 있을 것 같음
- **핵심 니즈:** 간편한 기록, 사회적 동기부여

### 페르소나 B — 목표 중심 절약가 "재테크 관심 대학원생"

- **나이/직업:** 28세, 대학원생
- **상황:** 매달 저축 목표가 있지만 카페·쇼핑 지출이 넘침
- **불편:** 지출 카테고리가 어디서 새는지 파악하기 어려움
- **기대:** 카테고리별 통계와 주간 리포트로 구체적인 개선 포인트 파악
- **핵심 니즈:** 통계·분석, 챌린지 달성률 추적

## 3. 사용자 흐름

```
[최초 진입]
온보딩 슬라이드 1 → 온보딩 슬라이드 2 → 로그인

[메인 루프]
홈 (대시보드)
 ├── 소비체크 탭
 │    ├── 오늘 지출 현황 조회
 │    ├── 소비 올리기 → 항목 추가 → 채팅방 공유(선택)
 │    └── 소비 통계 / 주간 리포트 조회
 ├── 챌린지 탭
 │    ├── 나의 챌린지 목록 → 챌린지 상세(참여 중)
 │    ├── 인기 챌린지 탐색 → 챌린지 상세(참여 전) → 참여하기
 │    └── 새 챌린지 만들기
 ├── 채팅 탭
 │    ├── 채팅 목록
 │    ├── 일반 채팅방
 │    └── 챌린지 채팅방 → 소비 공유 → 채팅방 선택(Checkbox)
 ├── 알림 탭
 │    └── 알림 목록 (D-day / 랭킹 / 미기록 / 리포트)
 └── 마이 탭
      ├── 프로필 / 소비 리스트 / 친구 관리
      ├── 설정 → 소비 기록 목록
      └── 챌린지 설정 → 랭킹
```

## 5. 화면별 상세 기능 명세

### 5.1 온보딩 (Onboarding)

**목적:** 앱의 핵심 가치를 2단계로 전달하고 로그인으로 유도

| 슬라이드 | 헤드라인 | 서브 메시지 | 비주얼 |
|----------|----------|-------------|--------|
| 1 | "무심코 나가는 소비,<br>생각보다 많지 않나요?" | 문제 인식 유도 | 소비 패턴 일러스트 |
| 2 | "기록하면 보이고,<br>줄이면 달라져요" | 솔루션 제시 | 챌린지 참여 일러스트 |

**UX 규칙:**
- 슬라이드 하단 페이지 인디케이터 (도트) 표시
- 스와이프 또는 하단 버튼으로 진행
- 2번 슬라이드에서 "시작하기" → 로그인 화면 이동
- 재진입 시 온보딩 스킵, 바로 홈으로 이동 (로그인 유지 상태)

---

### 5.2 로그인 (Login)

**목적:** 빠른 소셜 로그인 제공, 이메일 가입 보조 지원

**화면 구성:**

| 영역 | 내용 |
|------|------|
| 상단 로고 | MOTIMO 워드마크 |
| 슬로건 | "아끼고, 기록하고, 함께 바꾸는 소비 습관 챌린지" — Body2/Medium/16, Gray50 |
| 소셜 로그인 버튼 (3종) | 카카오 / 네이버 / Apple — 각 브랜드 컬러 적용 |
| 구분선 | "또는" 텍스트 divider |
| 이메일 가입 버튼 | 외곽선 버튼 (Outlined) |
| 보조 링크 | 이메일 찾기 \| 비밀번호 찾기 — Body2/Medium/16, Gray40 |

**소셜 로그인 버튼 순서:** 카카오(최우선) → 네이버 → Apple  
**에러 처리:** 로그인 실패 시 토스트 메시지 "로그인에 실패했어요. 다시 시도해 주세요"

---

### 5.3 홈 (Home)

**목적:** 앱의 메인 대시보드. 절약 현황과 인기 챌린지를 한눈에 확인

**화면 구성:**

| 영역 | 컴포넌트 | 상세 |
|------|----------|------|
| 상단 헤더 | 앱 로고 + 알림 아이콘 | 알림 미읽음 시 Primary 색상 뱃지 |
| 절약 현황 카드 | 이번 주 절약 금액 | Title/SemiBold/36으로 금액 강조 표시<br>전주 대비 증감률 (Secondary 색으로 표시) |
| 빠른 액세스 CTA | "오늘 소비 체크하기" 버튼 | Primary 버튼, 전체 너비 |
| 인기 챌린지 섹션 | "지금 인기 챌린지 🔥" 타이틀 | Body1/SemiBold/20 |
| 챌린지 카드 리스트 | 세로 스크롤 카드 목록 | 카드 1개당: 챌린지명, 참여 인원, 조건, 기간, 참여 버튼 |

**인기 챌린지 카드 상세 스펙:**

```
┌─────────────────────────────────┐
│ 🛍 쇼핑 금지 챌린지              │ ← Body1/SemiBold/20
│ 52명이 함께하고 있어요!          │ ← Body2/Medium/16, Gray50
│ 조건: 1일 1회 소비 체크          │ ← Body2/Medium/16, Gray40
│ 26.04.06(월) ~ 04.27(월)        │ ← Body2/Medium/16, Gray40
│                    [참여하기 →]  │ ← Primary 텍스트 버튼
└─────────────────────────────────┘
```

**정렬 기준:** 실시간 참여자 수 내림차순

---

### 5.4 소비체크 (Check)

**목적:** 오늘 지출 현황 확인 및 소비 통계 조회

**화면 구성:**

| 영역 | 컴포넌트 | 상세 |
|------|----------|------|
| 오늘의 소비 | 총액 + 예산 피드백 | Body1/SemiBold/24으로 금액 표시<br>"훌륭해요! 예산보다 5% 절약 중" (Primary 색상 코멘트) |
| 소비 올리기 버튼 | 플로팅 또는 인라인 CTA | → 소비 올리기 화면 이동 |
| 카테고리별 통계 | 파이 차트 + 범례 | 식비 45% / 교통 30% / 의료 25% / 쇼핑 15%<br>각 슬라이스에 카테고리 색상 적용 |
| 주간 리포트 | 요일별 막대 그래프 | 최근 7일, X축: 요일, Y축: 지출 금액<br>최고 지출일 막대 Secondary 색상 강조 |
| 지출 항목 목록 | 날짜별 그룹 리스트 | 이모지 + 상호명 + 금액 + 카테고리 태그 |

**예산 피드백 메시지 규칙:**

| 상태 | 메시지 | 색상 |
|------|--------|------|
| 절약 중 (5% 이상) | "훌륭해요! 예산보다 {N}% 절약 중" | Primary |
| 예산 근접 (90% 이상) | "주의! 예산의 {N}%를 사용했어요" | Secondary |
| 예산 초과 | "예산을 {N}원 초과했어요" | Red (별도 에러 컬러) |

### 5.5 소비 올리기 (Add Spending)

**목적:** 지출 항목을 카테고리별로 기록

**화면 구성:**

| 필드 | 타입 | 상세 |
|------|------|------|
| 카테고리 선택 | 수평 스크롤 탭 | `전체` `식비` `교통` `의료` `쇼핑` `카페` `여가` `생활` `기타`<br>선택된 탭: Primary 배경, 나머지: Gray10 배경 |
| 상호명 | 텍스트 입력 | placeholder "어디서 소비했나요?" |
| 금액 | 숫자 입력 | 숫자 키패드 자동 호출, 원화(₩) prefix |
| 날짜/시간 | 날짜 피커 | 기본값: 현재 날짜 |
| 메모 (선택) | 텍스트 입력 | placeholder "메모를 남겨보세요" |
| 저장 버튼 | Primary CTA | "저장하기" — 전체 너비 |

**지출 항목 예시:**

| 이모지 | 상호명 | 금액 | 카테고리 |
|--------|--------|------|----------|
| ☕ | 스타벅스 | 6,100원 | 카페 |
| 🛍 | 무신사 | 62,280원 | 쇼핑 |
| 🎤 | 노래방 | 10,000원 | 여가 |
| 🍿 | CGV | 15,000원 | 여가 |
| 📦 | 쿠팡 | 9,800원 | 쇼핑 |

**저장 후 동작:** 토스트 "소비가 기록되었어요!" → 소비체크 화면으로 복귀  
**채팅방 공유 옵션:** 저장 완료 후 "채팅방에 공유할까요?" 바텀 시트 노출

### 5.6 챌린지 (Challenge)

**목적:** 참여 중인 챌린지 관리 및 신규 챌린지 탐색

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 상단 탭 | "나의 챌린지" / "탐색" 탭 전환 |
| 나의 챌린지 카드 | 챌린지명, 참여 인원, D-Day, 달성률 프로그레스 바, 기간 |
| 하단 CTA | "지금 참여하세요 🔥" 플로팅 버튼 → 챌린지 탐색 화면 |
| 새 챌린지 만들기 버튼 | 우상단 아이콘 또는 플로팅 (+) 버튼 |

**나의 챌린지 카드 상세 스펙:**

```
┌──────────────────────────────────┐
│ 카페 금지 챌린지 ☕    D-3        │ ← Secondary 색상 D-Day 뱃지
│ 83명 참여 중                      │
│ ████████░░ 70%                   │ ← Primary 프로그레스 바
│ 26.04.06 ~ 04.27                 │
└──────────────────────────────────┘
```

**D-Day 뱃지 규칙:**
- D-7 이상: Gray 뱃지
- D-3 이하: Secondary(주황) 뱃지
- D-1: 빨간 뱃지 + "내일 마감!" 텍스트

### 5.7 챌린지 상세 — 참여 전 (Challenge Detail — Join)

**목적:** 챌린지 정보 제공 및 참여 유도

| 항목 | 상세 |
|------|------|
| 챌린지명 | Body1/SemiBold/24, 이모지 포함 |
| 조건 | "1일 1회 소비 체크" |
| 기간 | 시작일 ~ 종료일 (총 N일간) |
| 참여 인원 | "총 N명이 함께하고 있어요!" — Primary 텍스트로 숫자 강조 |
| 챌린지 소개 | 자유 텍스트, 최대 200자, Body2/Medium/17 |
| 개설자 정보 | 프로필 이미지 + 닉네임 |
| CTA | "챌린지 참여하기" — Primary 버튼, 전체 너비, 하단 고정 |

**참여 완료 후:** 챌린지 채팅방 자동 입장 안내 바텀 시트 표시

|------|------|
| 챌린지명 | Body1/SemiBold/24, 이모지 포함 |
| 조건 / 기간 / 참여 인원 | 참여 전과 동일 구성 |
| 나의 참여도 | 원형 프로그레스 차트 + 달성률 % 텍스트<br>"🔥 오늘 소비 체크하면 {N}% 달성!" 동기부여 메시지 |
| 나의 소비 체크 | "7/10 완료" — 완료/전체 표시 |
| CTA — 미완료 | "오늘 소비 체크하기" — Primary 버튼 |
| CTA — 완료 | "오늘 완료 ✓" — 비활성(Disabled) 버튼 |
| 소비 랭킹 바로가기 | 우상단 또는 섹션 내 링크 → 랭킹 화면 |

### 5.9 새 챌린지 만들기 (New Challenge)

**목적:** 사용자가 직접 챌린지를 개설

**입력 필드:**

| 필드 | 타입 | 유효성 검사 |
|------|------|-------------|
| 챌린지 제목 | 텍스트 입력 | 최소 2자, 최대 20자. 공백만 입력 불가 |
| 목표 금액 | 숫자 입력 | 1,000원 단위 스텝, 최소 1,000원 |
| 기간 | 날짜 범위 피커 | 시작일 ≥ 오늘, 최소 3일, 최대 30일 |
| 챌린지 소개 | 텍스트에어리어 | 최소 10자, 최대 200자. 글자 수 카운터 표시 |
| 카테고리 | 단일 선택 칩 | `전체` `식비` `교통` `의료` `쇼핑` `카페` `여가` `생활` `기타` |

**저장 버튼:** 모든 필수 필드 입력 시 Primary 활성화, 미입력 시 Disabled  
**개설 완료 후:** 생성된 챌린지 상세 화면으로 이동 + "챌린지가 만들어졌어요!" 토스트

### 5.10 채팅 목록 (Chat)

**목적:** 참여 중인 채팅방 목록 조회

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 검색창 | "채팅방 또는 친구 검색" placeholder |
| 채팅방 목록 | 최근 메시지 시간 기준 내림차순 정렬 |

**채팅방 카드 스펙:**

```
┌──────────────────────────────────────┐
│ [썸네일] 소비 기록 챌린지 📝  16명    │ ← Body1/SemiBold/17
│          이번 달도 다 같이 살아남아… │ ← Body2/Medium/16, Gray50, 1줄 말줄임
│                            오후 8:45 │ ← Body2/Medium/16, Gray40
│                              [미읽음] │ ← Primary 원형 뱃지 + 숫자
└──────────────────────────────────────┘
```

**채팅방 유형 구분:**
- 챌린지 채팅방: 챌린지 아이콘 + 이모지 썸네일
- 일반 채팅방: 그룹 아이콘 썸네일

---

### 5.11 일반 채팅방 (Chatting)

**목적:** 그룹 멤버 간 자유 채팅

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 상단 헤더 | 채팅방 이름 + 인원 수 + 뒤로가기 + 메뉴 아이콘 |
| 메시지 목록 | 발신자명, 버블, 시간, 읽음 수 |
| 하단 입력창 | 텍스트 입력 + 전송 버튼 (Primary 색상) + 첨부 아이콘 |

**말풍선 규칙:**

| 구분 | 정렬 | 배경색 |
|------|------|--------|
| 내 메시지 | 우측 | Primary `#90D50F` |
| 상대 메시지 | 좌측 | Gray10 `#F6F6F6` |

**읽음 수:** 버블 하단 소형 텍스트로 미읽음 수 표시 (예: "46명 읽음")

### 5.12 챌린지 채팅방 (Chatting-Challenge)

**목적:** 챌린지 참여자 간 전용 채팅. 소비 공유 및 달성 현황 확인

**일반 채팅방 대비 추가 요소:**

| 요소 | 상세 |
|------|------|
| 챌린지 배너 | 채팅 목록 상단 고정. 챌린지명 + 참여 인원 + "달성 현황 보기" 링크 |
| 소비 공유 버튼 | 입력창 옆 아이콘 → 소비 내역 채팅방 공유 플로우 진입 |
| 공유된 소비 카드 | 채팅 내 카드 형태: 이모지 + 상호명 + 금액 + 카테고리 |
| 챌린지 인터랙션 | 하단 고정 영역: "오늘 소비 체크" 빠른 버튼 |

### 5.13 채팅방 공유 (Share / Checkbox)

**목적:** 소비 내역이나 챌린지를 특정 채팅방/친구에게 공유

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 검색창 | "공유하고 싶은 친구, 채팅방 검색" |
| 필터 탭 | `전체` \| `친구` \| `최근 공유` |
| 목록 | 채팅방/친구 이름 + 인원 수 + 체크박스 |
| 하단 CTA | "보내기" — 1개 이상 선택 시 Primary 활성화 |

**다중 선택:** 최대 5개 채팅방 동시 공유 가능

### 5.14 알림 (Notification)

**목적:** 챌린지 관련 알림 및 앱 활동 피드 제공

**알림 유형 상세:**

| 유형 | 메시지 예시 | 발송 조건 | 딥링크 |
|------|-------------|----------|--------|
| 챌린지 D-day | "소비 기록 챌린지 📝 D-1! 마지막까지 함께해요 💪" | 종료 D-3, D-1 | 챌린지 상세 |
| 랭킹 변동 | "🔥 순위가 올랐어요! 지금 5위예요" | 순위 변동 시 | 랭킹 화면 |
| 소비 미기록 | "아직 오늘 기록이 없어요 ✍️ 지금 체크하고 챌린지 이어가요!" | 당일 21시 미기록 시 | 소비체크 |
| 주간 리포트 | "이번 주 리포트가 도착했어요 📊 얼마나 절약했는지 확인해보세요!" | 매주 월요일 오전 9시 | 소비 통계 |
| 챌린지 완료 | "챌린지를 완료했어요 🎉 수고했어요!" | 챌린지 종료일 | 챌린지 상세 |

**알림 카드 스펙:**
- 아이콘 + 메시지 본문 + 경과 시간
- 미읽음 알림: Gray10 배경 강조
- 읽은 알림: White 배경

### 5.17 랭킹 (Ranking)

**목적:** 챌린지 내 참여자 순위 확인으로 경쟁 동기부여 제공

**화면 구성:**

| 영역 | 상세 |
|------|------|
| 이번 주 소비 체크 현황 | 요일별(월~일) 체크 완료 여부 표시 |
| 랭킹 목록 | 순위 + 프로필 이미지 + 닉네임 + 체크 수 |

**순위 뱃지:**
- 1위: 🥇 Gold
- 2위: 🥈 Silver  
- 3위: 🥉 Bronze
- 4위 이하: 숫자 순위 텍스트

**나의 순위 강조:** 나의 행은 Primary 연한 배경(`#90D50F` 10% opacity) 적용

**예시 순위:**

| 순위 | 닉네임 |
|------|--------|
| 🥇 1 | 하나 |
| 🥈 2 | 텀블러쓰기3일차 |
| 🥉 3 | 민지 |
| 4 | **나** |
| 5 | 지은 |
| 6 | 라떼말고물 |
